## **Topic - 2: Edge and Fog Computing**

### <u>Edge Computing</u>

- Data is processed **close to the source**, minimizing latency.
- Reduces cloud dependency for real-time decision-making.


### <u>Fog Computing</u>

- Intermediate layer between edge and cloud.
- Enables **distributed storage, computation, and analytics**.

| Aspect      | Edge         | Fog                    |
| ----------- | ------------ | ---------------------- |
| Location    | Device level | Gateway / Local server |
| Latency     | Ultra-low    | Moderate               |
| Scalability | Limited      | High                   |
