# $\fbox{Chapter 3: PERCENTAGE}$





## **Topic - 1: Repeated Patterns**

| Sno. | Type                                  | Formula / Trick                                                                          | Example                                                                             |
| :--: | ------------------------------------- | ---------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------- |
|  1   | $\%$ of a number                      | $(x\;\%\;of\;y) = (\frac{x}{100}) \times y$                                              | $25\;\%\;of\;400 = 100$                                                             |
|  2   | Number from $\%$                      | $(y\;\%\;of\;x = z)$ → $y = \frac{(z × 100)}{x}$                                         | $20\;\%\;of\;x = 80$ → $x = 400$                                                    |
|  3   | $\%$ increase / decrease              | $\frac{(new − old)}{old} \times 100$                                                     | $50 \rightarrow 65 = 30 \% \uparrow$                                                |
|  4   | Successive $\%$ change                | $x + y + \frac{xy}{100}$                                                                 | $20\% \uparrow$ then $10\% \uparrow\; = 20 + 10 + 2 = 32 \%$                        |
|  5   | $\%$ more / less comparison           | $A$ is $x\%$ more than $B \rightarrow A = \frac{B(100+x)}{100}$                          | $A$ is $25\%$ more than $B = 1.25 B$                                                |
|  6   | Reverse percentage                    | If after $x\%$ increase price becomes $y$ → original $= y \times \frac{100}{(100+x)}$    | After $25\% \uparrow$ price $= 500$ → original $= 500 \times \frac{100}{125} = 400$ |
|  7   | Fraction ↔ $\%$ conversion (memorise) | $\frac{1}{4} = 25\%$, $\frac{1}{5} = 20\%$, $\frac{1}{8} = 12.5\%$, $\frac{1}{20} = 5\%$ | $\frac{3}{8} = 37.5\%$                                                              |
|  8   | $\%$ of $\%$ trick                    | $x\%\;of\;y\%\;of\;z = \frac{(x \times y \times z)}{10000}$                              | $20\%\;of\;30\%\;of\;400 = \frac{(20 \times 30 \times 400)}{10000} = 24$            |



## **Topic - 2: Percentage Shortcut**

| Situation                            | Shortcut                       |
| ------------------------------------ | ------------------------------ |
| $x\%\;of\;y = y\%\;of\;x$            | Same value                     |
| $10\%$ → Move decimal one place left | $456.7 \times 10\% = 45.67$    |
| $5\%$ → Half of $10\%$               | $340 \times 5\% = 17$          |
| $1\%$ → Divide by $100$              | $2500 \times 1\% = 25$         |
| $25\%$ → Divide by $4$               | $800 ÷ 4 = 200$                |
| $12.5\%$ → Divide by $8$             | $640 ÷ 8 = 80$                 |
| $33 \frac{1}{3}\%$ → Divide by $3$   | $90 ÷ 3 = 30$                  |
| $66 \frac{2}{3}\%$ → $\frac{2}{3}$   | $240 \times \frac{2}{3} = 160$ |
| $16 \frac{2}{3}\%$ → $\frac{1}{6}$   | $300 ÷ 6 = 50$                 |
| $37.5\%$ → $\frac{3}{8}$             | $400 \times \frac{3}{3} = 150$ |
