# $\fbox{Chapter 6: INTERESTS}$





## **Topic - 1: Simple Interest (SI)**

| Formula                                | What it means                                 |
| -------------------------------------- | --------------------------------------------- |
| $SI = \frac{P \times R \times T}{100}$ | $P$ = Principal, $R$ = Rate %, $T$ = Time yr. |
| $Amount = P + SI$                      | Total money after time                        |
| $P = \frac{100 \times SI}{R \times T}$ | Find principal                                |
| $R = \frac{100 \times SI}{P \times T}$ | Find rate                                     |
| $T = \frac{100 \times SI}{P \times R}$ | Find time                                     |



## **Topic - 2: Compound Interest (CI)**

### <u>Formulae</u>

| Formula                              | AFCAT only asks for $n = 2$ or $3$ yrs |
| ------------------------------------ | -------------------------------------- |
| $Amount = P (1 + \frac{r}{100})^n$   | Final amount                           |
| $CI = Amount − P$                    | Interest                               |
| $CI = P [(1 + \frac{r}{100}^n) − 1]$ | Direct CI formula                      |
