# $\fbox{Chapter 10: NUMBER SYSTEM}$





## **Topic - 1: Repeated Patterns**

| Type                   | Formula / Trick                                            | Example                                                |
| ---------------------- | ---------------------------------------------------------- | ------------------------------------------------------ |
| Divisibility Rules     | Memorise $2–13$ rules                                      | $123456$ divisible by $3$?<br>Sum $21 \rightarrow$ yes |
| HCF & LCM              | $HCF \times LCM = \text{Product of numbers}$               | HCF $12$ & $18$ = $6$,<br>LCM = $36$                   |
| HCF by Euclidean       | $HCF(a,b) = HCF(b,\;a\;mod\;b)$                            | HCF $48$ & $18$ = $HCF(18,12)$ = $HCF(12,6)$ = $6$     |
| Factors & Multiples    | Number of factors = $(p+1)(q+1)(r+1)$ for $p^q \times r^s$ | $36 = 2^2 × 3^2 \rightarrow$ factors $(2+1)(2+1) = 9$  |
| Unit Digit / Cyclicity | Last digit cycle: $2(4)$, $3(4)$, $7(4)$, $8(4)$, $9(2)$   | $7^5$ unit digit = $7$                                 |
| Remainders             | $(a + b) mod\;m = (a\;mod\;m + b\;mod \;m) mod\;m$         | $25 + 17\;mod\;7$<br>$= 4 + 3 = 7\;mod\;7 = 0$         |
| Prime/ Composite/ Test | Prime test up to $\sqrt n$                                 | $97$ prime (check up to $9$)                           |
| Successive Division    | For $3$ numbers HCF/LCM                                    | HCF $12,18,30 = 6$                                     |



## **Topic - 2: Tricks / Shortcuts**

| Situation               | Shortcut                                                                        |
| ----------------------- | ------------------------------------------------------------------------------- |
| Divisibility by $7$     | Take last digit, double, subtract from rest                                     |
| Divisibility by $11$    | Alternate sum                                                                   |
| Unit digit cycle        | $2,3,7,8$ cycle $4$; $4$ cycle $2$; $9$ cycle $2$; $5$&$6$ always $\frac{5}{6}$ |
| HCF of $3$ numbers      | $HCF(HCF(a,b),c)$                                                               |
| LCM of $3$ numbers      | $LCM(LCM(a,b),c)$                                                               |
| Number of factors       | Prime factorize → (exponents + 1) product                                       |
| Number of prime factors | Count distinct primes                                                           |
| Co-prime                | $HCF = 1$                                                                       |
| Remainder trick         | $(a \times b) mod\;m = (a\;mod\;m \times b\;mod\;m) mod\;m$                     |
| Successive remainder    | For large powers                                                                |
| Even/Odd                | Last digit even/odd                                                             |
| Square ending           | $0,1,4,5,6,9$ only                                                              |
