# $\fbox{Chapter 8: TIME, DISTANCE \& DISTANCE}$





## **Topic - 1: Repeated Patterns**

| Type                                    | Formula / Shortcut                                                                                                                    | Example                                                                                   |
| --------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| Basic                                   | $\text{Speed} = \frac{\text{Distance}}{\text{Time}}$,<br>$v=\frac{s}{t}$                                                              | $60\frac{km}{h} \times 3h = 180km$                                                        |
| Average Speed (two equal dist)          | $\frac{2ab}{a+b}$ (harmonic mean)                                                                                                     | $40\;\&\;60\frac{km}{h} \rightarrow 2 \times 40 \times \frac{60}{40+60} = 48\frac{km}{h}$ |
| Relative Speed (same direction)         | $A - B$                                                                                                                               | $70-50=20\frac{km}{h}$                                                                    |
| Relative Speed (opposite)               | $A + B$                                                                                                                               | $70 + 50 = 120 \frac{km}{h}$                                                              |
| Boat & Stream                           | $\text{Downstream} = b + s$,<br>$\text{Upstream} = b − s$,<br>$\text{Boat speed} = \frac{d+u}{2}$,<br>$\text{Stream} = \frac{d−u}{2}$ | $d = 20$,<br>$u = 12 \rightarrow boat = 16$,<br>$stream = 4$                              |
| Train crossing (pole/man)               | $\text{Time} = \frac{\text{Length of train}}{\text{Relative speed}}$                                                                  | $200m$ train at $72 \frac{km}{h} \rightarrow 10sec$                                       |
| Train crossing (platform/another train) | $t = \frac{L_1 + L_2}{\text{Relative speed}}$                                                                                         | Two $150 m$ trains opposite $\rightarrow \frac{300}{Relative}$                            |



## **Topic - 2: Shortcuts / Tricks**

| Situation                                 | Shortcut                                                |
| ----------------------------------------- | ------------------------------------------------------- |
| $\frac{km}{h} \rightarrow \frac{m}{s}$    | $\times \frac{5}{18}$                                   |
| $\frac{m}{s} \rightarrow \frac{km}{h}$    | $\times \frac{18}{5}$                                   |
| Avg. speed (equal distance)               | $\frac{2ab}{a+b}$                                       |
| $\text{Time ratio = Inverse speed ratio}$ | Distance same                                           |
| $\text{Distance ratio = Speed ratio}$     | Time same                                               |
| Boat still water                          | $\frac{down + up}{2}$                                   |
| Stream speed                              | $\frac{down − up}{2}$                                   |
| Train length from pole time               | $Speed \times Time$                                     |
| Train crossing platform                   | $\frac{\text{Train + Platform}}{\text{Relative speed}}$ |
| Two objects opposite                      | $\text{Relative speed = sum}$                           |
| Same direction                            | $\text{Relative speed = difference}$                    |
| $A$ beats $B$ by $x$ m in race of $y$ m   | $\text{Speed ratio} = y:(y−x)$                          |
