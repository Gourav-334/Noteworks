# $\fbox{Chapter 5: PROFIT \& LOSS}$





## **Topic - 1: Question Varieties**

| Type                    | Formula / Trick                                                                   | Example                                                                      |
| ----------------------- | --------------------------------------------------------------------------------- | ---------------------------------------------------------------------------- |
| Basic P/L               | $Profit = SP − CP$, $Loss = CP − SP$                                              | $CP=800, SP=1000 \rightarrow Profit\;200$                                    |
| % Profit / Loss         | $\% = \frac{P}{CP} \text{ or } \frac{L}{CP} \times 100$                           | $Profit=200 \text{ on } CP=800 \rightarrow 25 \%$                            |
| SP from %               | $SP = CP \times \frac{100 ± \%}{100}$                                             | $CP=500, 20\%\;profit \rightarrow SP=600$                                    |
| CP from %               | $CP = SP \times \frac{100}{100 ± \%}$                                             | $SP=600, 20\%\;profit \rightarrow CP=500$                                    |
| Successive %            | Same as percentage chapter: $x + y + \frac{xy}{100}$                              | $10\% \text{ profit then } 20\%\text{ profit} = 32\%$                        |
| Dishonest dealer        | Uses wrong weight $\rightarrow \% profit = \frac{true − false}{false} \times 100$ | Sells $900 g$ for $1 kg \rightarrow \frac{100}{9}\%\;profit \approx 11.11\%$ |
| Marked Price & Discount | $SP = MP \times \frac{100 − d\%}{100}$                                            | $MP=1200, 25\% \text{ discount} \rightarrow SP=900$                          |



## **Topic - 2: Shortcuts & Tricks**

| Situation                                    | Trick                                      |
| -------------------------------------------- | ------------------------------------------ |
| $\text{\% Profit on CP = \% Loss on SP}$     | Same numerical value                       |
| Two successive equal %                       | $Net = 2x + \frac{x^2}{100}$               |
| $100\%$ profit                               | $SP = 2 \times CP$                         |
| $100\%$ loss                                 | $SP = 0$                                   |
| Dishonest dealer ($x g$ instead of $1000 g$) | $Profit\% = \frac{1000 − x}{x} \times 100$ |
| Marked price trick                           | Always calculate discount on MP            |
| SP same, % profit different                  | CP inverse to profit %                     |
| $Profit\% = \frac{SP}{CP − 1} \times 100$    | Direct formula                             |
| $Loss\% = \frac{1 − SP}{CP} \times 100$      | Direct formula                             |
| $SP = CP \times \frac{100 + p}{100}$         | Memorize for 10%, 20%, 25%, 50%            |
