# $\fbox{Chapter 4: RATIOS \& PROPORTIONS}$





## **Topic - 1: Formulae**

| Type                        | Formula / Trick                                                                | Example                                                                             |
| :-------------------------- | ------------------------------------------------------------------------------ | ----------------------------------------------------------------------------------- |
| Simple Ratio                | $a:b \rightarrow a/b$                                                          | $3:4 = 3/4$                                                                         |
| Divide in ratio             | $Part_1 = \frac{a}{a+b} \times total$<br>$Part_2 = \frac{b}{a+b} \times total$ | $Rs. 700$ in $3:4 \rightarrow 300\;\&\;400$                                         |
| Compound Ratio              | $(a \times c \times e) : (b \times d \times f)$                                | $2:3, 4:5, 5:6 \rightarrow 40:90 = 4:9$                                             |
| Proportion ($a:b = c:d$)    | $a \times d = b \times c$ (cross multiply)                                     | $2:3 = 4:x \rightarrow x = 6$                                                       |
| Continued Proportion (mean) | $b^2 = a \times c \rightarrow b = \sqrt(a×c)$                                  | $4, x, 9 \rightarrow x = 6$                                                         |
| Ratio after adding/removing | $\text{New ratio = (old part ± change)}$                                       | $5:7, \text{add } 3 \text{ to each} \rightarrow 8:10 = 4:5$                         |
| Partnership / Age / Mixture | Ratio of investment / time / quantity                                          | $A:B\;3:4,$<br>$\text{profit } 700 \rightarrow \text{ A gets } 300, \text{ B } 400$ |



## **Topic - 2: Shortcuts**

| Situation                                 | Trick                                                          |
| ----------------------------------------- | -------------------------------------------------------------- |
| Divide amount in ratio m:n                | $Part = \frac{m}{m+n} \times total$                            |
| Ratio after adding same to both           | New ratio $= (a+k):(b+k)$                                      |
| Ratio after subtracting                   | $(a−k):(b−k)$                                                  |
| Inverse ratio                             | $b:a$                                                          |
| Ratio of speeds = ratio of time (inverse) | $\text{Distance same } \rightarrow \text{ time ratio inverse}$ |
| Ages ratio after $t$ years                | Add $t$ to each part                                           |
| Partnership profit ratio                  | $= \text{investment } \times \text{ time ratio}$               |
| Mixture alligation                        | $\text{(cheaper − mixture) : (mixture − costly)}$              |
| Mean proportional                         | $\sqrt(first \times third)$                                    |
| Duplicate/Triplicate ratio                | $(a^2:b^2)\;or\;(a^3:b^3)$                                     |
