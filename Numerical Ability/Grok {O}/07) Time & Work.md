# $\fbox{Chapter 7: TIME \& WORK}$





## **Topic - 1: Repeated Patterns**

| Type                                | Formula / Trick                                        | Example                                                                                                                                                                           |
| ----------------------------------- | ------------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Individual work rate                | $\text{1 day work = } \frac{1}{\text{Total days}}$     | $A$ finishes in $10$ days → $\frac{1}{10}$ per day                                                                                                                                |
| Combined work rate                  | $A + B = \frac{1}{a} + \frac{1}{b}$                    | $A=10 \text{ days},$<br>$B=15 \text{ days} \rightarrow \frac{1}{10} + \frac{1}{15} = \frac{1}{6} \rightarrow 6 \text{ days together}$                                             |
| Time taken together                 | $\frac{1}{\frac{1}{a} + \frac{1}{b}} = \frac{ab}{a+b}$ | $10 \& 15 \rightarrow 10 \times \frac{15}{10+15} = 6\;days$                                                                                                                       |
| Work left / extra                   | $\text{Work left = 1 − Work done}$                     | $A \& B \text{ do } \frac{5}{6} \rightarrow \frac{1}{6} \text{ left}$                                                                                                             |
| $A$ alone after $B$ worked $x$ days | $B$’s $x$ days work + $A$’s remaining work             | $B \text{ works } 3 \text{ days} \rightarrow \frac{3}{15} = \frac{1}{5},$<br>$\text{ remaining } \frac{4}{5} \rightarrow A \text{ takes } \frac{4}{5} \times 10 = 8 \text{ days}$ |
| Pipes & Cisterns                    | Inlet = +ve, Outlet = −ve                              | $\text{Inlet } \frac{1}{10},$<br>$\text{ outlet } \frac{1}{15} \rightarrow \frac{3}{30} − \frac{2}{30} = \frac{1}{30} \rightarrow 30 \text{ days to fill}$                        |
| Work efficiency ratio               | Efficiency inverse to time                             | $\text{Time } 2:3 \rightarrow \text{ efficiency } 3:2$                                                                                                                            |



## **Topic - 2: Shortcuts / Tricks**

| Situation                                              | Shortcut                                               |
| ------------------------------------------------------ | ------------------------------------------------------ |
| $A$ alone after $B$ worked $x$ days                    | $B$’s work in $x$ days + $A$’s remaining work          |
| $A$ & $B$ together $n$ days, $A$ alone $m$ days        | $B \text{ alone} = \frac{mn}{m−n}$                     |
| Ratio of time → inverse efficiency                     | $\text{Time } 2:3 \rightarrow \text{ efficiency } 3:2$ |
| Pipes: filling time                                    | $\frac{1}{\frac{1}{a} − \frac{1}{b}}$                  |
| Pipes: emptying time                                   | $\frac{1}{\frac{1}{b} − \frac{1}{a}}$                  |
| Work equivalence                                       | $M_1D_1 = M_2D_2$                                      |
| Men & Women ratio                                      | Convert to same unit first                             |
| $A$ does $\frac{1}{k}$ work, $B$ does rest in $y$ days | $A$’s time = $k y$                                     |
| Negative work (leak/outlet)                            | Subtract rates                                         |
| LCM method for days                                    | Take LCM of individual days $\rightarrow$ total work   |
