# TCS IPA UI Cheat Sheet

> Focus: Basic User Interface concepts, Web fundamentals, HTML, CSS, JavaScript, UX, and common UI terminology.

---

# 1. What is UI?

UI (User Interface) is the medium through which a user interacts with software.

#### Examples:

- Buttons
- Menus
- Text fields
- Checkboxes
- Icons

---

# 2. UI vs UX

| UI | UX |
|----|----|
| User Interface | User Experience |
| Appearance | Overall experience |
| Visual design | Ease of use |
| Colors, fonts | User satisfaction |

---

- Radio Button
- Checkbox

- Submit Button

# 6. CSS

- CSS = Cascading Style Sheets

#### Purpose:

- Styling webpages

#### Example:

```css
h1{
    color:blue;
}
```

---

# 7. Ways to Apply CSS

1. Inline

```html
<p style="color:red;">
```

2. Internal

```html
<style>
</style>
```

3. External

```html
<link rel="stylesheet">
```

#### Preferred:

✅ External CSS
