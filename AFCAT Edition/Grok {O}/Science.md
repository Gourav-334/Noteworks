## **Topic - 1: Formulae**

| Physics Formula / Value   | Value / Formula                       |
| ------------------------- | ------------------------------------- |
| Speed ($v$)               | $\frac{s}{t}$                         |
| Acceleration ($a$)        | $\frac{v-u}{t}$                       |
| 1st equation of motion    | $v = u + at$                          |
| 2nd equation of motion    | $s=ut+\frac{at^{2}}{2}$               |
| 3rd equation of motion    | $2as=v^{2}-u^{2}$                     |
| Force ($F$)               | $F = ma$                              |
| Weight ($w$)              | $mg$                                  |
| $g$ on earth              | $9.8$ m/s²                            |
| Work ($W$)                | $F \times s \times cos \theta$        |
| Power ($P$)               | $\frac{W}{t}$, $F \times E$           |
| Kinetic Energy ($K.E.$)   | $\frac{mv^{2}}{2}$                    |
| Potential Energy ($P.E.$) | $mgh$                                 |
| Density ($D$)             | $\frac{m}{V}$                         |
| Pressure                  | $\frac{F}{A}$                         |
| Pressure in liquid        | $\rho gh$                             |
| Buoyant force             | Weight of displaced liquid            |
| Speed of sound in air     | ≈ $330\;or\;340$ m/s                  |
| Speed of light            | $3 \times 10^{8}$ m/s                 |
| Focal length (mirror)     | $f = \frac{R}{2}$                     |
| Lens formula              | $\frac{1}{f}=\frac{1}{v}-\frac{1}{u}$ |
| Power of lens             | $P=\frac{1}{f}$                       |
| Ohm’s Law                 | $V = IR$                              |
| Power (electric)          | $P=VI=I^{2}R=\frac{V^{2}}{R}$         |
